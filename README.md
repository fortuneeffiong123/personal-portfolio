🌐 **A Personal Portfolio Website**
A responsive personal portfolio website showcasing my skills, projects, and experience as a Cloud/DevOps Engineer. The project is built using HTML, CSS, and JavaScript and deployed using Netlify for fast and reliable hosting.


🚀 **Live Demo Link**
👉 https://fortune-effiong-portfolio.netlify.app

📌 **Features**
*Responsive design (mobile, tablet, desktop)
*Modern UI/UX layout
*About section with personal details
*Projects showcase section
*Skills and technologies overview
*Contact information section
*Fast deployment via Netlify

**🛠️ Built With**
*HTML5
*CSS3
*JavaScript
*Netlify (Hosting & Deployment)
*Git & GitHub (Version Control)
